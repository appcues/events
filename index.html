Static Showdown Boston

January 24 - 25 @ hack/reduce in Kendall Square | Organized by Appcues


Register in ~17 seconds

The easiest way to get there is to take the MBTA Red Line to Kendall and walk up Third street.


Static Showdown is a 48-hour global hackathon featuring static web apps. It was created by Divshot, and is sponsored by companies like Dropbox, GitHub, Codeship, Firebase, and Appcues.

You can learn more about the worldwide Static Showdown here. To save your spot for the Boston hub, register on the left.
